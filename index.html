FAQ

Uma bot do Discord é um programa que pode ser adicionado a um servidor do Discord para adicionar funcionalidades específicas, como comandos de economia, utilidade e diversão.

Para adicionar a bot ao seu servidor do Discord, é necessário ter permissões de administrador. Em seguida, você pode usar o botão "Adicionar a Bot" no ínicio do site.

Sim, a bot é segura de usar. A única informação que coletamos são os comandos executados, usuário, hora e servidor esses dados são coletados para fins de evitar problemas. Além disso, nosso bot é open source, garantindo transparência e segurança adicionais.

Para usar os comandos da bot, digite o prefixo do bot seguido do nome do comando desejado no canal de texto do Discord. O prefixo padrão da Nyssa bot é "/".

Sim, a bot fica online 24 horas por dia, 7 dias por semana para garantir que seus comandos estejam sempre disponíveis.

Se você encontrar algum problema com a bot, pode entrar em contato com o desenvolvedor pelo Discord ou pelo GitHub para relatar o problema e obter ajuda. Ou usar o comando "/problema".
